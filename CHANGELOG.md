# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-04-09

### Changed
- Moved support button to the top of the admin settings page
- Removed margin-top/padding-top/border-top CSS from the support button section

## [1.0.0] - Initial tracked release

### Added
- Animated thumbnail image slider for discussion list
- Automatic image extraction from first post content
- Smooth fade + scale animation with CSS transitions
- Lazy loading with IntersectionObserver
- Smart layout integration with topic-rating, magnet-link, and discussion-views extensions
- Responsive design with mobile scaling
- Dark mode support
- Configurable image dimension filters
- Admin panel settings for slider width, autoplay speed, max images, and dimension limits
